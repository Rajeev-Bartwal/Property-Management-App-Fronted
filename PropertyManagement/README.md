# Mini Property App – Frontend

This is the frontend of the **Mini Property App**, built using **React.js with Vite**.  
It provides a simple UI to view properties, add new properties, and see property details.

---

## Tech Stack
- React.js
- Vite
- React Router DOM
- JavaScript
- CSS

---

## Features
- Dummy Login (Email & Password validation)
- Property List view
- Add Property form
- Property Detail view
- Dummy data support for testing
- Ready for Spring Boot backend integration
